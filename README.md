# 🚀 CodeAlpha Python Programming Internship Tasks

Welcome to my **CodeAlpha Internship** project repository. This repository contains the Python projects I completed during my internship, showcasing my programming skills, problem-solving abilities, and practical implementation of Python concepts.

---

## 📂 Projects

### 🤖 1. AI Chatbot

A simple rule-based chatbot developed using Python and the NLTK library. The chatbot can respond to greetings, answer common questions, tell jokes, and maintain basic conversations using predefined patterns.

#### Features
- Interactive text-based conversation
- Pattern matching using NLTK
- Multiple predefined responses
- Beginner-friendly implementation

#### Technologies Used
- Python
- NLTK

---

### 📈 2. Stock Portfolio Tracker

A Python application that helps users track their stock investments by fetching real-time stock prices and calculating portfolio value and profit/loss.

#### Features
- Add and remove stocks
- Fetch live stock prices
- Calculate portfolio value
- Display profit/loss
- Tabular output for better readability

#### Technologies Used
- Python
- yfinance
- tabulate

## ▶️ Getting Started

### Clone the Repository

```bash

```

### Navigate to the Project

```bash
cd codealpha_tasks
```

### Install Dependencies

```bash
pip install nltk yfinance tabulate
```
